从头开始：执行顺序

有了新的 ODB 文件，按以下顺序跑：

1
配置提取范围，跑提取层
在配置表里登记要提取的 case 和 frame，然后跑两个提取脚本
python scripts/extracted/extract_load_from_odb.py python scripts/extracted/extract_stress_from_odb.py
2
跑插值主链（自动编排）
在 interpolation_cases.csv 里登记 case 和目标载荷，跑编排入口。它会自动完成 clean → 帧选择 → raw 插值 → smooth 平滑。
python scripts/analysis/interpolation/prepare_interpolation_cases.py
3
跑 load 分析（可与步骤 2 并行）
直接读 extracted/load，不依赖步骤 2，提取完成后随时可跑
python scripts/analysis/load/load_slope_change_analysis.py
4
跑 K 链（步骤 2 完成后）
在 k_tasks.csv 和 k_stage_params.csv 里配置任务，按顺序跑四个脚本
python scripts/analysis/k/k_fe_stress_fitting_analysis.py python scripts/analysis/k/export_k_tables.py python scripts/analysis/k/eval_rstar_and_margin.py # 按需 python scripts/analysis/k/rstar_scan.py # 按需
