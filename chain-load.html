UF 曲线 slope / slope_change 计算 · 提取层后的并列下游
这条链做什么

直接读取提取层的标准 UF 曲线，计算 slope 和 slope_change，输出 features 结果。

本链是提取层之后的并列下游之一，不依赖插值链，提取完成后随时可跑。

跑完 extract_load_from_odb.py 之后，这条链就可以独立执行，无需等待插值链完成。
脚本
load_slope_change_analysis.py
scripts/analysis/load/load_slope_change_analysis.py
扫描 extracted/load/ 下的各 case，读取 UF 曲线，计算 slope 和 slope_change，结果写入 analysis/load/<case>/features/。
python scripts/analysis/load/load_slope_change_analysis.py
输入 / 输出
上游输入
extracted/load/<case>/<case>_UF_curve.csv
唯一正式输入，不依赖 analysis/interpolation
下游输出
analysis/load/<case>/features/<case>_UF_with_slope_change.csv analysis/load/logs/*.txt
不允许将旧目录中的中间结果作为正式输入。唯一正式输入是 extracted/load/ 下的标准 UF 曲线。
职责边界

本链负责：标准 UF 曲线读取、slope / slope_change 计算、结果落盘、日志记录。

不负责：ODB 提取、stress 处理、interpolation、plotting 展示扩展、K 链计算。
